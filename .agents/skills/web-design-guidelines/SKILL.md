# Web Design Guidelines

## Objetivo

Garantir que o produto final seja funcional, intuitivo e com estética premium.

## Como Usar

Ao desenhar ou implementar novas telas:

### 1. Funcionalidade Primeiro

- Uma interface bonita que não funciona é uma falha. Priorize a clareza das ações do usuário.
- **Hierarquia Visual**: Use tamanho, contraste e espaçamento para guiar o olho para os elementos mais importantes.

### 2. Consistência e UX

- **Mobile-First**: O design deve funcionar perfeitamente em telas menores antes de expandir.
- **Consistência**: Mantenha tipografia, paleta de cores e estilos de componentes uniformes em todo o site.
- **Navegação Clara**: Use menus simples e caminhos de navegação (breadcrumbs) visíveis.

### 3. Performance e Acessibilidade

- **Fast Loading**: Otimize imagens e use lazy-loading para mídias pesadas.
- **Acessibilidade (A11y)**: Use contraste alto, labels ARIA e garanta que todas as ações sejam acessíveis via teclado.
