Contributing to VANTAGE
========================

Thank you for your interest in contributing. VANTAGE is released under AGPLv3 — all
contributions to the public core remain open and auditable.

Contribution focus
------------------

Good fits for this repository:

* Bug fixes and UX improvements in the public core
* Tests, documentation, and release hardening
* Feed, recon, watchlist, shift handoff, analysis, and admin improvements that belong in the public product

Out of scope:

* Contract-specific deployments
* Managed-service internals
* Premium extension code
