# `web-legacy/`

Este diretório existe apenas como **arquivo histórico** do frontend anterior do VANTAGE.

## Status

- não é mais fonte de verdade
- não participa do runtime oficial
- não é usado por Docker, CI ou build principal
- deve ser consultado apenas para referência histórica pontual

## Fonte oficial atual

Use [`../web`](../web) como frontend canônico do produto.

## O que ainda tem valor aqui

- textos e nomenclaturas antigas para comparação editorial
- testes do shell anterior como referência histórica
- implementação anterior para auditoria ou comparação de comportamento

## O que não deve ser feito

- não adicionar novas features aqui
- não corrigir bugs aqui como fluxo principal
- não apontar documentação operacional para este diretório

Quando o valor histórico remanescente deixar de ser útil, este diretório poderá ser removido ou exportado para arquivo externo.
